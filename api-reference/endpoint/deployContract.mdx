---
title: "deploySmartContract"
description: "Deploys a new smart contract to the network."
---

**Route**
```typescript
POST /api/contracts
```

**Parameters**
```typescript
{
  "name": string,
  "owner": string,
  "version": string,
  "code": string,
  "type": string,
  "state": object
}
```

**Returns**
```typescript
{
  "success": boolean,
  "address": string  // Contract address
}
```

**Example - cURL**
```bash
curl -X POST https://blockexplorer.replit.app/api/contracts \
  -H "Content-Type: application/json" \
  -d '{
    "name": "MyToken",
    "owner": "0xownerABC",
    "version": "1.0.0",
    "code": "class Contract { transfer(to, amount) { /* logic */ } }",
    "type": "ERC20",
    "state": {}
  }'
```

**Example - Typescript**
```typescript
import axios from "axios";

async function deploySmartContract() {
  const response = await axios.post("https://blockexplorer.replit.app/api/contracts", {
    name: "MyToken",
    owner: "0xownerABC",
    version: "1.0.0",
    code: "class Contract { transfer(to, amount) { /* logic */ } }",
    type: "ERC20",
    state: {},
  });

  console.log("Contract Address:", response.data.address);
}

deploySmartContract();

```
