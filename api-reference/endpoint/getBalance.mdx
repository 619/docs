---
title: "getBalance"
description: "Retrieves the balance of a given wallet address."
---

**Route**
```typescript
GET /api/wallets/:address
```

**Returns**
```typescript
{
  "balance": number
}
```

**Example - cURL**
```bash
curl -X GET https://blockexplorer.replit.app/api/wallets/0xuserABC
```

**Example - cURL**
```typescript
import axios from "axios";

async function getBalance(address: string) {
  const response = await axios.get(`https://blockexplorer.replit.app/api/wallets/${address}`);
  console.log("Balance:", response.data.balance);
}

getBalance("0xuserABC");
```
