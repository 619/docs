---
title: "sendTransaction"
description: "Broadcasts a signed transaction to the network."
---

**Route**
```typescript
POST /api/transactions
```

**Parameters**
```typescript
{
  "type": "transfer",
  "sender": string,
  "data": {
    "recipient": string,
    "amount": number
  }
}
```

**Returns**
```typescript
{
  "success": boolean,
  "id": string  // Transaction ID
}
```

**Example - cURL**
```bash
curl -X POST https://yourdomain.com/api/transactions \
  -H "Content-Type: application/json" \
  -d '{
    "type": "transfer",
    "sender": "0x123abcDEF456",
    "data": {
      "recipient": "0x456defABC789",
      "amount": 100
    }
  }'

{
  "success": true,
  "id": "tx_12345abcde"
}
```

**Example - Typescript**
```typescript
import axios from "axios";

async function sendTransaction() {
  const response = await axios.post("https://yourdomain.com/api/transactions", {
    type: "transfer",
    sender: "0x123abcDEF456",
    data: {
      recipient: "0x456defABC789",
      amount: 100,
    },
  });

  console.log("Transaction ID:", response.data.id);
}

sendTransaction();
```
