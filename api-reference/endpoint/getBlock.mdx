# getBlock

Returns details about a specific block.

**Parameters**
```typescript
{
  blockNumberOrHash: string | number  // Block identifier
}
```

**Returns**
```typescript
{
  number: number,
  timestamp: Date,
  transactions: Array<{
    id: string,
    type: string,
    sender: string,
    status: string
  }>
}
```
