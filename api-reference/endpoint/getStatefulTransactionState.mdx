---
title: "getStatefulTransactionState"
description: "Retrieves current state of a stateful transaction channel."
---

**Parameters**
```typescript
{
  channelId: string  // Channel ID
}
```

**Returns**
```typescript
{
  status: 'active' | 'finalized' | 'cancelled',
  currentState: any,
  lastUpdateTime: Date,
  participants: string[],
  stateHistory: Array<{
    state: any,
    timestamp: Date,
    updatedBy: string
  }>
}
```
