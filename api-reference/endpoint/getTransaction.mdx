---
title: "getTransaction"
description: "Retrieves details of a specific transaction."
---

**Route**
```typescript
GET /api/transactions/:id
```

**Returns**
```typescript
{
  "id": string,
  "timestamp": string,
  "type": string,
  "sender": string,
  "data": any,
  "status": string,
  "error": string | null,
  "stateChanges": Array<{ key: string, oldValue: any, newValue: any }>
}
```

**Example - cURL***
```typescript
curl -X GET https://blockexplorer.replit.app/api/transactions/tx_12345
```

**Example - Typescript**
```typescript
import axios from "axios";

async function getTransaction(transactionId: string) {
  const response = await axios.get(`https://blockexplorer.replit.app/api/transactions/${transactionId}`);
  console.log("Transaction Details:", response.data);
}

getTransaction("tx_12345");
```
