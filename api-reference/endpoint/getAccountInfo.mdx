---
title: "getAccountInfo"
description: "Retrieves high-level information about a wallet, including recent transactions."
---

**Route**
```typescript
GET /api/account/:address
```

**Parameters**
```typescript
{
  address: string  // Address to look up
}
```

**Returns**
```typescript
{
  "address": string,
  "balance": number,
  "nonce": number,
  "recentTransactions": Array<{
    "id": string,
    "timestamp": string,
    "type": string,
    "sender": string,
    "status": string,
    "error"?: string,
    "data": any,
    "state_changes": any
  }>
}
```

**Example**
```typescript
# Example using curl
curl -X GET https://uranusrpc.replit.com/api/account/0x123abcDEF456

# JSON response:
{
  "address": "0x123abcDEF456",
  "balance": 5000,
  "nonce": 3,
  "recentTransactions": [
    {
      "id": "abc123...",
      "timestamp": "2025-02-01T12:34:56.789Z",
      "type": "transfer",
      "sender": "0x123abcDEF456",
      "status": "pending",
      "data": { "to": "0x456def...", "amount": "0x1f4" },
      "state_changes": {}
    },
    ...
  ]
}
```
