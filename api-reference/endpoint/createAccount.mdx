---
title: "createAccount"
description: "Generates a new wallet with a unique address and private key."
---

**Route**
```typescript
POST /api/create-account
```

**Parameters**
```typescript
None
```

**Returns**
```typescript
{
  "address": string,
  "privateKey": string
}
```

**Example**
```typescript
# Example using curl
curl -X POST https://uranusrpc.replit.com/api/create-account

# JSON response:
{
  "address": "bfb64e2ad4fe66cf2af37ca5d6db0a7ea54f65c2",
  "privateKey": "f8ac2b0d0fa2d1b4aff6..."
}
```
