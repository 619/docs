---
title: "createAccount"
description: "Creates a new wallet account on the blockchain. Optionally returns a private key if requested."
---

## API Details

```typescript
POST /api/account/create

// Response
{
  address: string,
  privateKey?: string
}
```

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `returnPrivateKey` | `boolean` | No | If true, includes the private key in the response. Defaults to false. |

## Returns

| Field | Type | Description |
|-------|------|-------------|
| `address` | `string` | The newly created wallet address |
| `privateKey` | `string` | (Optional) The private key for the wallet. Only included if returnPrivateKey is true |

## Example Request

```typescript
const response = await fetch('https://api.yourchain.com/api/account/create', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer your-api-key',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    returnPrivateKey: true
  })
});

const account = await response.json();
// {
//   address: "0x1234567890abcdef1234567890abcdef12345678",
//   privateKey: "0xabc123..." // Only if returnPrivateKey was true
// }
```
